Flutter app Psell

This is our academic Project 2, a group project where we created a Psell app for plant sale shop owners and their customers. We developed this app using Flutter and Firebase technologies. I contributed to various parts of the project.

  • QR Generator and Scanner
  • Admin Panel Plant Upload
  • Homepage Retrieval
  • Admin Bill System
  • User Management in Admin Panel
  • Session Timeout
  • Consultancy Booking
  • Add to Cart
  • Pre-order Bookings
  • Add a New Admin

Our comprehensive plant sale app, Psell, offers a user-friendly interface and a range of features catering to both shop owners and customers. The combination of Flutter and Firebase technologies ensures a robust and efficient platform for plant enthusiasts.
